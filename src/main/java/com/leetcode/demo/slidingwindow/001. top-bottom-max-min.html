Mastering Java Streams

20 Programming Challenges for Top/Bottom & Max/Min Patterns

Level 1

Basic (Primitives & Strings)

01. Top 3 Numbers Find the 3 largest integers in a list.
02. Smallest 5 Find the 5 smallest doubles in a list.
03. Longest String Find the single longest string in a list of names.
04. Bottom 3 Alphabetical Get the first 3 strings when sorted alphabetically.
05. Unique Top 5 Get the 5 highest unique numbers, effectively handling duplicates.
Level 2

Intermediate (Custom Classes)

Context: Use a Product(name, price, rating) or Employee(name, age, salary) class for these challenges.
06. Highest Paid Find the Employee with the maximum salary.
07. Top 5 Earners List the top 5 Employee objects by salary.
08. Youngest Employee Find the Employee with the minimum age.
09. Budget Picks Get the 10 cheapest Product objects.
10. Premium Products Get the 3 products with the highest rating.
11. Alphabetical First Find the Employee whose name comes first in the dictionary.
12. Most Recent Hire If Employee has a LocalDate hireDate, find the person hired last.
Level 3

Advanced (Logic & Edge Cases)

13. Second Highest Find the 2nd highest salary (Tip: use skip(1)).
14. Top 3 by Name Length Find the 3 employees with the longest names.
15. Lowest Non-Zero Find the lowest price that is strictly greater than zero.
16. Tied Max Find the oldest person; if ages are equal, pick the one with the higher salary.
17. Top 5 Distinct Prices Get the top 5 most expensive products, but ensure no two products have the same price.
18. Top 10% In a list of 100 items, find the top 10 using dynamic limiting.
19. Min Rating with Stock Find the lowest-rated product that is currently "in stock" (use filter).
20. Last 3 Alphabetical Find the 3 names that appear last in alphabetical order, returned in descending order.
Level 4

Group By Pattern

Key Tool: Collectors.groupingBy(). Think of this as the SQL GROUP BY clause for Java collections.
21. Departmental Directory Group a list of Employee objects by their department.
22. Item Frequency Count the number of times each String appears in a list (e.g., Apple: 3, Banana: 2).
23. Average Salary by Dept Group employees by department and calculate the average salary for each.
24. Status Mapping Group Order objects by status, but the map value should be a List<Long> of order IDs only.
25. Top Performer per Dept Group employees by department and find the highest-paid person in each group.
26. Price Summary Group Product objects by category and return DoubleSummaryStatistics (sum, min, max, avg) for each.
27. Partitioning Even/Odd Divide a list of integers into two groups: one for even numbers and one for odd numbers.
28. Group by Name Length Given a list of names, group them by the number of characters they contain.
29. Revenue by Region Group SalesRecord objects by region and calculate the total revenue (sum) for each region.
30. Multi-Level Grouping Group employees by department, and then further group them by jobRole within that department.
31. Name Concatenation Group employees by department and return a Map where the value is a Comma-Separated String of all names in that dept.
32. Score Classification Classify a list of integers into "Pass" (>=60) or "Fail" (<60) and store the results in a Map<String, List<Integer>>.
33. Unique Values per Group Group Employee objects by department, but ensure the resulting values are a Set of Names to remove duplicates.
34. Boolean Counting Partition a list of employees by their "Active" status and return a Map<Boolean, Long> containing the count of each.
35. Most Frequent Character Find the character with the highest frequency in a given String using grouping and sorting.
36. Monthly Transaction Total Group Transactions by their Month and calculate the total amount spent per month.
37. Conditional Grouping (Filtering) Group employees by department, but only include those with more than 5 years of service in the lists.
38. Alphabetical TreeMap Group items by category and ensure the result is stored in a TreeMap so keys are automatically sorted.
39. Youngest per Department Group employees by department and return the single youngest employee for each group.
40. Nested Average Calculation Group orders by Year and then Quarter, calculating the Average Order Price for each nested group.
41. Group by Custom Logic Group a list of words by their first character, converting the keys to uppercase.
42. Salary Range Buckets Group employees into salary buckets: "Low" (<30k), "Mid" (30k-70k), and "High" (>70k).
43. Multi-Value Mapping Group Students by Grade, but map the values to a List containing only their Phone Numbers.
44. Partitioning with Summing Partition a list of Invoices into "Paid" and "Unpaid" and calculate the total sum for each partition.
45. Grouping by Multiple Fields Create a key by concatenating department and city, then group employees by this composite key.
46. Most Populated Category Group Products by category and return only the name of the category that has the most items.
47. Collectors.teeing Example Group a list of numbers and find both the sum and average in a single grouping operation.
48. Grouping Null Values Group a list of Strings, but ensure that any null values are collected into a group with the key "UNKNOWN".
49. Date-Based Partitioning Partition a list of Events into "Upcoming" and "Past" based on the current system date.
50. Reducing inside Groups Group employees by department and use reducing to find the total number of years the group has worked combined.
